About Me

Hello! My name is Matthew Turner, I'm a 24-year-old self-taught Python programmer, and I created the site you are currently on! I designed this site, and the projects within, as a way to practice my Python coding abilities.

My journey began in June of 2019 when I took a job working at Regeneron Pharmaceuticalâ€™s Global Security Operations Command Center as a Security Analyst. After working for a few months, I noticed that there were a few responsibilities assigned to me that had the potential to be automated. At this point in my life, the only coding experience I had was some limited experience in Java from high school. After some recommendations from friends, and looking around on the internet, I decided to try installing Python and started teaching myself the language.

For the next 2 years I continued to teach myself Python. I wrote basics scripts that automated some of my repetitive responsibilities, and this opened up time for me to work on more important projects and responsibilities. Over time this brought recognition from my superiors, who asked if I could assist in managing the Security database and other Access Control related tasks using the scripts I was writing. For the rest of my time with Regeneron I assisted in their Access Control operations using the Python abilities I had taught myself. I even had a chance to pitch a project to the Director and manager of Security; I created a script that would take all Regeneron Employee Travelers flight numbers for the day and input and display it on a flight tracking program called FlightAware. FlightAware would then display any flights currently in the air with Regeneron travelers. The program was so well liked by the Director and Manager that they even put down money to pay for a premium FlightAware account.

Moving forward, I'm hoping to one day become a Software Developer or Data Scientist. I'm continuing to teach myself Python, which has ignited my passion for coding. Please feel free to look around the site and see the results of my efforts. If you would like to ask any questions, or just want to reach out, feel free to fill out a contact form.
